## Purpose

A live, always-on **lobby dashboard for EQC Institute / Equinim College's Perth Campus**. It runs on a Samsung Frame TV in reception and shows current room allocations, the day's class schedule, weather, upcoming events, the campus floorplan, a Google Map of the surrounding area, and a scrolling announcements banner. Trainers sign on for their class via a QR-coded portal (which also writes to a Google Form / Apps Script log), and admin staff manage rooms, events and announcements through a hidden settings panel — all updates push live to every connected screen via WebSockets.

## Target audience

The dashboard is read by everyone who enters the lobby:

- **Students** — checking which room their class is in, scanning the WiFi QR, seeing what's coming up
- **Trainers and staff** — using the sign-on portal at the start of their day, glancing at room status
- **Visitors and prospective students** — getting a first impression of the campus, scanning course-info QRs

Designed for **passive glanceability** first, **interactive use** second.

## Primary user journey

1. A visitor walks into the EQC Perth lobby and sees the Samsung Frame screen with current rooms, time, weather, and any active announcements
2. A trainer scans the QR code (or taps the cog) to sign on for their class — the room flips to "Live" with course/topic/intake/trainer details across every screen in real time
3. Admin staff log into the hidden settings panel to add events, post announcements, or override room status
4. At midnight the system auto-resets all rooms to "available" and clears the day's staff sign-ons

## Core features

- Live room allocation grid with `available` / `live` / `break` states and animated status indicators
- Trainer sign-on portal at `/trainer-sign-on.html` (Google Form embed)
- Admin panel with tabs for Rooms, Events and Announcements (password-protected)
- Real-time updates across all connected clients via Socket.io
- Marquee announcement banner (multiple severities/colours, auto-expires)
- Perth weather forecast widget (currently mock data — ready for an API swap)
- Campus floorplan tile + embedded Google Map of the surrounding area
- Bottom info row: Study Hub QR, WiFi QR (`EQC-network`), Courses link, Instagram
- Footer with campus address, phone, email, fire-assembly point, first-aid, RTO/CRICOS numbers
- Fullscreen toggle for kiosk display
- Daily auto-reset at midnight

## Tech stack

- **Frontend:** React 19 + TypeScript + Vite 6, Tailwind CSS 4, Motion (animations), Lucide icons, qrcode.react
- **Backend:** Express 4 + Socket.io 4 (single Node process serves both the SPA and the WS)
- **Build / runtime:** `tsx` for the server, Vite for the client; `npm run start` boots the production server
- **Hosting target:** Google Cloud Run (containerised, scales to zero, supports WebSockets)
- **Domain references:** `equinimcollege.com`, `study.equinimcollege.com`
